# Task 14: GitHub Actions Docker Build Workflow

## Objective
Create `.github/workflows/docker.yml` — a GitHub Actions workflow that builds the Docker image on pushes to main/master.

## Deliverables

### File Created: `.github/workflows/docker.yml`
- Location: `.github/workflows/docker.yml`
- Triggers: `on: push: branches: [main, master]` (push only, no PRs)
- Single job: `docker-build`
  - Runs on: `ubuntu-latest`
  - Steps:
    1. Checkout code (actions/checkout@v4)
    2. Build Docker image: `docker build -t gitignore-lol .`

### YAML Validation
✓ File structure is valid YAML
✓ Proper indentation and syntax
✓ No registry push, no secrets, no multi-platform builds
✓ Triggers only on push to main/master branches

### File Content
```yaml
name: Docker Build

on:
  push:
    branches:
      - main
      - master

jobs:
  docker-build:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout code
        uses: actions/checkout@v4

      - name: Build Docker image
        run: docker build -t gitignore-lol .
```
